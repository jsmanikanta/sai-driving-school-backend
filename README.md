# Sai Driving School Backend

A backend system built using Node.js, Express.js, and MongoDB to manage services for a driving school and travel platform. This backend handles user interactions, car listings, bookings, and administrative operations.

## Features

- RESTful API architecture  
- User data handling and form submissions  
- Car listing and management system  
- Booking/request handling  
- Admin panel support (approve/reject/manage data)  
- MongoDB database integration  
- Structured backend with controllers and routes
